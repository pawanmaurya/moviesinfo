Movies Info
Get IMDB info for your movies collection in 2 clicks
(only Google chrome supported)
Select Movies Folder


If movie collection is in D:\my_data\movies select `movies` directory, NOT D:

How it works

Video files are searched on the basis of file extension. Then using Tmdb api movie information is populated in datatable. Movies are sorted by Imdb ratings.

Nothing is uploaded to server. We don't store your movie collection information.
